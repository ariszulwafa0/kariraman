Pendahuluan

Selamat datang di dokumentasi API KarirAman. API ini memungkinkan Anda untuk mengintegrasikan logika verifikasi lowongan kerja cerdas kami ke dalam aplikasi atau layanan Anda.

Base URL

Semua endpoint API dimulai dengan URL dasar yang sama. Gunakan URL Vercel Anda setelah deploy.

https://kariraman.vercel.app

Autentikasi

Saat ini, API bersifat terbuka. Di masa depan, mungkin akan diperlukan API Key yang dikirim melalui header `Authorization`.

POST /api/cek-teks

Menganalisis teks lowongan kerja untuk mendeteksi tanda-tanda penipuan dan memberikan penilaian risiko.

Request Body

{
    "text": "Teks lowongan kerja lengkap..."
}

POST /api/cek-rekening

Memeriksa nomor rekening di database resmi Kominfo (cekrekening.id) untuk riwayat laporan penipuan.

Request Body

{
    "bank": "bca",
    "account": "1234567890"
}

Untuk nama bank, gunakan singkatan umum (misal: bca, bri, bni, mandiri, bsi, cimb, jenius, jago, dll).
